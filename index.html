    Doubly Linked List – Theory

    Definition

    A doubly linked list is a linear data structure in which each node contains data, a pointer to the previous node, and a pointer to the next node.

    Advantages

    • Bidirectional traversal
    • Efficient insertion and deletion
    • Suitable for navigation-based applications

    Disadvantages

    • Extra memory overhead
    • More complex pointer handling

    Why suitable for music playlists?

    Music players require frequent next and previous navigation, dynamic reordering, and real-time updates — all of which are naturally supported by doubly linked lists.
